A substrate ready Docker image based on theia.

## Dev

```
nvm use 10
yarn
yarn dev
```

## Build

```
nvm use 10
yarn
yarn build
```

## Docker image

Run the image via:

```
docker run -p 80:3000 -p 8000:8000 -p 9944:9944 gcr.io/substrateplayground-252112/jeluard/theia-substrate@sha256:XXX
```
